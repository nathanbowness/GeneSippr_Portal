Welcome to the GeneSippr Portal!

Information

This webtool allows users to run Genesippr - a linux command line tool - from inside a website.

A pair of files in .fastq.gz format, can be uploaded to the website where a project may be created. The project should include the files you would like to run a Genesippr task on, however many projects can be created.

Upon the Genesippr task completing the results may be viewed under the 'GeneSippr Results' side bar.
